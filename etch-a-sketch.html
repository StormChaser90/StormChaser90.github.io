Etch-A-Sketch

How to Use

  1. Drag the slider to set the size of the grid.
  2. Click "Set"
  3. Select color
  4. Draw away!

Tips:

  • After selecting a color of your choice, you must close the color wheel in order to apply the selection.
  • Your drawing may included multiple colors, simply select a new color as you go!
